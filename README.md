# PGLife

A full-stack web project built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
- XAMPP (Apache + MySQL)

## Setup Instructions
1. Install XAMPP
2. Move the project folder to `htdocs`
3. Start Apache and MySQL
4. Import the database from `database/pglife.sql` using phpMyAdmin
5. Open `http://localhost/PGLife`
